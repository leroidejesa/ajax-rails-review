# Personal Portfolio version 0.0.1

Completed as an assignment at [Epicodus](www.epicodus.com), Portland, Oregon.

Made with Ruby on Rails.

### Setup
1. Clone Repo
`$ git clone https://github.com/leroidejesa/rails-wk1-review.git`

2. Create Database
`$ rake db:setup`

3. Run PostgreSQL
`$ postgres`

4. Run Server
`$ rails s`

5. Visit `localhost:3000` in your web browser.
